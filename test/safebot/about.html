About SafeBot

Features

Pokémon Collecting

Catch and collect 150+ unique Pokémon!

Different Pokémon spawn in different environments, allowing users to find their favourites!

Rarity of Pokémon

Every Pokémon that spawns in a given Spawn Area is given a rarity, which defines the chances of finding that Pokémon. The chances of each rarity spawning are as follows:

Common Uncommon Rare Very rare Super rare Legendary Mythical
50% 20% 15% 8.5% 5.5% 0.9% 0.1%

These chances may change during certain events or as part of future updates.

Shiny Pokémon

Every Pokémon users encounter can be shiny!

The chances of finding a shiny Pokémon vary based on the rarity of the Pokémon.

Marked Pokémon

Marked Pokémon are a super-rare form of Pokémon, which can be shown off to friends!

Every Pokémon has a 1/1500 chance to be Marked.

Quests

Quests are difficult tasks that users can complete to earn points, exclusive Pokémon, and more!

Spawn Events

There is often an active Spawn Event.

Spawn Events are when the Event Spawn Area has a special assortment of Pokémon able to be found.

Spawn Events often increase the chances of finding shiny Pokémon, and can make it easier to find rare or new Pokémon!

Economy System

Earn points and spend them at the store

Earning Points

Users can earn points through three methods:

Method Points Earned Description
Daily points 350 Claim free points every 24 hours
Working 80-120 Claim points every two hours
Giveaways - Win a points giveaway in the Safe Haven Discord server
Quests - Complete quests to earn points

Spending Points

Users can spend their points in the store.

The store contains rare and exclusive Pokémon that you can add to your collection.

Points Balance

Users can check their own balance, or that of another user.

Fun & Games

Fun commands to enjoy

Spotify Search

Searches Spotify's API for the song that best matches a user's search query and provides details about it.

Random Colour Generation

Generates a random colour.

Also generates an image with the colour in order to show users how it looks.

Provides RGB and Hex values.

About User

Provides details about a user provided by the Discord API.

No personal information is provided.

About Server

Provides details about the current server provided by the Discord API.

Magic 8ball

Ask the magic 8ball a question.

Coinflip

Flip a coin, it will land on heads or tails.

Ticketing

Create private channels to communicate with staff

Creating the GUI

Users with the Manage server permission can create the 'open ticket' GUI.

Opening a Ticket

Users can click the "open ticket" button user the 'open ticket' GUI

Adding Users to the Ticket

Users can add other users to a ticket.

Removing Users from the Ticket

Users with the Kick users permission can remove non-staff users from the ticket.

Closing the Ticket

Users can close the ticket via the button at the start of the ticket or by running the close ticket command.
